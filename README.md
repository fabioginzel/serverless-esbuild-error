## Desenvolvimento

* Baixar projeto do git

* Montar os dockers
```sh
make build
```

* Start nas maquinas
```sh
make start
```

* Rodar os testes
```sh
make test 
```

* Parar as maquinas
```sh
make stop
```

* Entrar na maquina de desenvolvimento
```sh
make bash-server
```
